In this project we perform an Exploratory Data Analysis (EDA) on a dataset of 114,000 Spotify tracks sourced from Kaggle.

Objective: Analyze Spotify streaming trends to uncover patterns in music consumption.

ðŸ’¡ Key columns in this dataset include:

* track_id: The unique Spotify ID for each track.<br>
* artists: Names of the artists who performed the track, separated by ';'.<br>
* album_name: The name of the album in which the track appears.<br>
* track_name: The title of the track.<br>
* popularity: A value between 0 and 100, indicating the track's popularity based on recent plays.<br>
* explicit: Boolean indicating whether the track contains explicit content.<br>
* danceability: Describes how suitable a track is for dancing (0.0 = least danceable, 1.0 = most danceable).<br>
* energy: Represents the intensity and activity of a track (0.0 = low energy, 1.0 = high energy).<br>
* loudness: Overall loudness of the track in decibels (dB).<br>
* valence: Measures the musical positiveness conveyed by a track (0.0 = negative, 1.0 = positive).<br>
* tempo: Estimated tempo of the track in beats per minute (BPM).<br>
